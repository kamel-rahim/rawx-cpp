#rawx-cpp

## This is a rewritting of the rawx service of the oio-sds project.
Using C++ and Facebook proxygen.

## Third-party dependencies
* [Gtest]
* [Glog]
* [Gflag]
* [Proxygen]
* [folly]
* [Wangle]

## Installation
   ```
   $ git clone https://github.com/kamel-rahim/rawx-cpp.git
   $ cd rawx-cpp
   $ cmake -DSYS=OFF .
   $ make
   ```
